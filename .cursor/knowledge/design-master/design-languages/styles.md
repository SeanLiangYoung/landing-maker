# 设计风格知识

## 极简风格（Minimal Style）

### 核心特征
- **简洁**：去除不必要的装饰元素
- **留白**：充足的留白空间，创造呼吸感
- **功能性**：形式服从功能
- **有限配色**：通常使用 1-3 种颜色

### 依赖知识
- 留白（generous-whitespace）
- 无衬线字体（sans-serif-fonts）
- 有限配色（limited-color-palette）

### 应用场景
- 追求简洁、专业、现代感的产品
- 科技产品、SaaS 产品
- 高端品牌

### 参考案例
- Apple 官网
- Google Material Design
- Stripe 官网

### 设计建议
- 区块间距：80-120px
- 卡片内边距：32-48px
- 使用系统字体或现代无衬线字体
- 避免过多装饰元素

## 温暖友好风格（Warm Friendly Style）

### 核心特征
- **温暖**：使用温暖的色调
- **友好**：圆角元素、柔和阴影
- **亲和**：创造亲近感

### 依赖知识
- 温暖配色（warm-color-palette）
- 圆角元素（rounded-corners）
- 柔和阴影（soft-shadows）

### 应用场景
- 面向消费者的产品
- 需要亲和力的产品
- 教育、健康、生活服务类产品

### 参考案例
- Notion 官网
- Figma 官网
- Airbnb 官网

### 设计建议
- 使用温暖的色调（米色、浅棕、暖灰）
- 按钮、卡片使用圆角（8-16px）
- 使用柔和的阴影，避免过于强烈
- 保持专业性的同时增加亲和力

## 专业可信风格（Professional Trustworthy Style）

### 核心特征
- **专业**：结构化布局、专业字体
- **可信**：保守配色、清晰信息层次
- **权威**：严谨的设计规范

### 依赖知识
- 结构化布局（structured-layout）
- 专业字体（professional-typography）
- 保守配色（conservative-colors）

### 应用场景
- B2B 产品
- 金融科技
- 企业服务
- 需要建立信任的产品

### 参考案例
- Stripe 官网
- Linear 官网
- Salesforce 官网

### 设计建议
- 使用网格系统，保持一致的间距
- 使用专业、易读的字体
- 避免过于鲜艳的颜色
- 信息层次清晰，结构严谨

---

## 现代科技风格（Modern Tech Style）

### 核心特征
- **现代**：渐变、动画、现代视觉效果
- **科技感**：几何图形、数据可视化
- **创新**：突破常规的布局和交互

### 依赖知识
- 渐变背景（gradient-background）
- 平滑动画（animation-smooth）
- 功能网格（feature-grid）

### 应用场景
- 科技产品
- 创新产品
- 需要展示技术实力的产品

### 参考案例
- Vercel 官网
- GitHub 官网
- Linear 官网

### 设计建议
- 使用渐变背景增加视觉吸引力
- 添加平滑的过渡动画
- 使用几何图形和数据可视化
- 保持现代感和科技感
