# AI 简历优化工具 Landing 页面设计规格

**设计路径**：路径 3（参考+创意）  
**参考来源**：夸克 AI 简历  
**设计风格**：极简、现代、科技感、炫酷  
**创建时间**：2026-02-10

---

## 一、设计概述

### 1.1 设计目标

- **风格定位**：极简、现代、科技感、炫酷
- **情绪基调**：专业、可信、创新、友好
- **目标用户**：应届毕业生、跳槽/转行的专业人士
- **核心价值**：提高简历通过率，JD 定向优化，免费试用

### 1.2 设计原则

- **参考知识库**：
  - `minimal-style`（极简风格）：简洁、留白、功能性
  - `professional-trustworthy`（专业可信）：结构化布局、专业字体、保守配色
  - `feature-modular-display`（功能模块化展示）：多个功能独立成模块
  - `gradient-background`（渐变背景）：增加视觉吸引力
  - `animation-smooth`（平滑动画）：提升交互体验

---

## 二、整体布局

### 2.1 页面结构

1. **Hero 区块**（居中布局）
2. **数据展示区块**（新增，建立信任）
3. **功能展示区块**（功能模块化展示）
4. **案例展示区块**（新增，用户故事+数据成果+前后对比）
5. **定价展示区块**（卡片布局）
6. **隐私保护区块**（独立区块，突出显示）
7. **页脚**（公司信息、联系方式、法律链接）

### 2.2 布局系统

- **容器最大宽度**：1200px（桌面端）
- **网格系统**：12 列网格，列间距 32px
- **响应式断点**：
  - 移动端：< 768px（1 列）
  - 平板：768px - 1024px（2 列）
  - 桌面端：> 1024px（3-4 列）

## 四、字体系统

### 4.1 字体选择

**参考知识库**：`sans-serif-fonts`（无衬线字体）、`professional-typography`（专业字体）

- **主字体**：`Inter`（现代、专业、易读）
- **备用字体**：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`
- **代码字体**：`"SF Mono", Monaco, "Courier New", monospace`（如需要）

### 4.2 字体大小

- **H1（Hero 主标题）**：`64px` / `4rem`（桌面端），`48px` / `3rem`（移动端）
- **H2（区块标题）**：`40px` / `2.5rem`（桌面端），`32px` / `2rem`（移动端）

- **正文**：`16px` / `1rem`
- **小字**：`14px` / `0.875rem`

### 4.3 字重

- **H1/H2**：`700`（Bold）
- **H3/H4**：`600`（Semi-bold）
- **正文**：`400`（Regular）
- **强调文字**：`500`（Medium）

### 4.4 行高

- **标题行高**：`1.2`（紧凑）
- **正文行高**：`1.6-1.8`（舒适阅读）
- **大段落行高**：`1.8`（长文本）

## 七、动效设计

### 7.1 过渡动画

**参考知识库**：`animation-smooth`（平滑动画）、`smooth-transition`（平滑过渡）

- **过渡时长**：`150-300ms`
- **缓动函数**：`ease` 或 `cubic-bezier(0.4, 0, 0.2, 1)`
- **应用元素**：按钮、卡片、链接等所有交互元素

### 7.2 悬停效果

**参考知识库**：`hover-effect`（悬停效果）

- **卡片悬停**：`transform: translateY(-8px)`，阴影增强
- **按钮悬停**：`transform: translateY(-2px)`，阴影增强
- **链接悬停**：颜色变化，下划线动画

### 7.3 滚动动画

**参考知识库**：`scroll-animation`（滚动动画）

- **淡入动画**：元素进入视口时淡入
- **上滑动画**：元素从下方滑入
- **缩放动画**：元素轻微缩放
- **触发方式**：使用 Intersection Observer API

### 7.4 加载动画

**参考知识库**：`loading-animation`（加载动画）

- **页面加载**：淡入效果
- **数据加载**：骨架屏或加载指示器
- **图片加载**：懒加载 + 淡入效果

### 7.5 特殊动效

- **Hero 背景**：渐变背景可以使用缓慢的动画效果
- **数据计数**：数字从 0 开始递增到目标值
- **图标动画**：悬停时图标轻微旋转或缩放
- **粒子效果**：Hero 区块可以使用粒子效果（可选）
